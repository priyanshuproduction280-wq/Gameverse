# **App Name**: GamerVerse

## Core Features:

- Homepage Showcase: Display featured games with smooth animations and a premium UI.
- Game Listings: Publicly accessible grid layout of all available games with key details and hover animations.
- Game Detail Pages: Dedicated pages for each game with full descriptions, media, and purchase options.
- User Authentication: Firebase Authentication with Google and Email/Password options.
- Persistent Cart: Firestore-based cart system to save and retrieve users cart, available after login.
- Checkout and Payment: Display payment instructions and dynamically fetch the QR code.
- Admin Panel: Secure admin panel with the capability to add/edit/remove games, update the payment QR code, view all orders, and update order statuses.

## Style Guidelines:

- Primary color: Deep indigo (#4F3A65) for a premium and trustworthy feel.
- Background color: Dark slate gray (#212529) for a modern, high-end aesthetic.
- Accent color: Vibrant purple (#A772FB) for highlights and interactive elements.
- Headline font: 'Space Grotesk' sans-serif for headings and CTAs, matched with 'Inter' sans-serif for body text.
- Clean, modern icons for categories and interactive elements.
- Grid-based layout with glassmorphism cards, soft shadows, and staggered entrance animations on scroll.
- Smooth Framer Motion animations for page transitions, hover effects, and cart drawer animations.
